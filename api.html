API Reference

Defines

DEBUG

Define this macro to enable debug logging macros.

DBG_BUFFER_SIZE

Total number of bytes allocated for the debug log buffer.

DBG_LINE_MAX

Maximum length of a single formatted log line (excluding newline).

dbg_info(fmt, ...)

Log an informational message.

Parameters:

  • fmt – Format string

  • ... – Format arguments

dbg_warn(fmt, ...)

Log a warning message.

Parameters:

  • fmt – Format string

  • ... – Format arguments

dbg_error(fmt, ...)

Log an error message.

Parameters:

  • fmt – Format string

  • ... – Format arguments

dbg_trace(fmt, ...)

Log a trace-level message (for verbose function-level tracing).

Parameters:

  • fmt – Format string

  • ... – Format arguments

dbg_assert(expr)

Assert that an expression is true, otherwise halt.

Writes an error to the debug log, then triggers a halt (ESTOP0).

Parameters:

  • expr – Expression to evaluate

dbg_enter()

Mark function entry in the debug log.

dbg_exit()

Mark function exit in the debug log.

Functions

void dbg_init(void)

Initialise the debug log buffer.

This function zeroes out the internal buffer and resets the write index.

void dbg_log(const char *fmt, ...)

Write a formatted message to the debug log.

Parameters:

  • fmt – Format string (printf-style)

  • ... – Format arguments

void dbg_clear(void)

Clear the debug log buffer.

Alias for dbg_init(); resets the buffer to all zero and the write index to 0.

const char *dbg_get_buffer(void)

Retrieve a pointer to the debug buffer for inspection.

Returns:

Pointer to static character buffer holding the log.

size_t dbg_get_index(void)

Get the current write index in the debug log.

Returns:

Index (in bytes) of the last written log line.

Internal implementation of RAM-based debug logging for embedded systems.
